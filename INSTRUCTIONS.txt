TO INSTALL ON LOCAL COMPUTER: 
*some commands may need to be changed on Windows
1. open terminal and cd into location of project: e.g. cd path/to/project
2. activate virtual environment: python3 -m venv venv
3. install requirements: pip install -r requirements.txt
4. activate virtual environment: source venv/bin/activate
5. run flask web application: flask run
6. go to browser at: 127.0.0.1:5000 or localhost:5000
